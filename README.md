# Student Management System

## 📌 Project Description
This is a web-based Student Management System developed using Flask and SQLite. It allows users to add, view, update, and delete student records efficiently.

## 🚀 Features
- Add new student details
- View all student records
- Update student information
- Delete student records

## 🛠️ Technologies Used
- Python (Flask)
- HTML
- SQLite

## 📂 Project Structure
- app.py → Main backend application
- templates/ → HTML files
- students.db → Database file
